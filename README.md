# Rock Paper Scissors Game

## Description  
A simple console-based Rock Paper Scissors game developed in C++.  
The game allows the user to play against the computer with random moves.

## Technologies Used  
- C++  
- Standard input/output (console)  
- Random number generation  

## Features  
- User vs Computer gameplay  
- Input validation  
- Score tracking  
- Simple and interactive console interface  

## How to Run  
1. Compile the code using any C++ compiler (e.g. g++, Visual Studio)  
2. Run the executable  
3. Follow on-screen instructions to play
